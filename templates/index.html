Project Details

Train Seat Booking is a comprehensive application designed to facilitate efficient and user-friendly reservation of train seats. Built with Python Flask for the backend and Angular for the frontend, it ensures a seamless experience for both users and administrators.

Methodology

Utilizing a RESTful API architecture, the application integrates Flask-Cors for cross-origin requests and Gunicorn as the WSGI server. The frontend leverages Bootstrap for responsive design and Font Awesome for intuitive icons, ensuring a modern and accessible interface.
